## Getting Started

### Prerequisites

This project requires the following dependencies:

- TypeScript
- npm or Yarn

### Installation

1. **Clone the repository**  
   ```bash
   git clone https://github.com/gqtpe/spotify
   ```

2. **Navigate to the project directory**  
   ```bash
   cd spotify
   ```

3. **Install dependencies**

   Using npm:  
   ```bash
   npm install
   ```

   Using Yarn:  
   ```bash
   yarn install
   ```

---

## Usage

**Start the development server**

Using npm:  
```bash
npm start
```

Using Yarn:  
```bash
yarn start
```

---

**Build the project**

Using npm:  
```bash
npm run build
```

Using Yarn:  
```bash
yarn build
```

---

**Preview the production build**

Using npm:  
```bash
npm run preview
```

Using Yarn:  
```bash
yarn preview
```

---

**Run Storybook**

Using npm:  
```bash
npm run storybook
```

Using Yarn:  
```bash
yarn storybook
```

---

**Lint the code**

Using npm:  
```bash
npm run lint
```

Using Yarn:  
```bash
yarn lint
```

---

## Testing
